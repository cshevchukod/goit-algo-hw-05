# Висновки по ДЗ: Алгоритми пошуку (Task 3)

Тести робились на двох текстах (`стаття 1.txt`, `стаття 2 (1).txt`) та для двох підрядків:

- **існуючий**: `бінарний пошук`
- **вигаданий**: `суперсекретнийпідрядок12345`

Алгоритми:
- **Boyer–Moore**
- **KMP (Кнут–Морріс–Пратт)**
- **Rabin–Karp**

## Висновки

- Для **стаття 1** найшвидший: **Boyer–Moore**.
- Для **стаття 2** найшвидший: **Boyer–Moore**.
- Загалом по двох текстах теж лідер: **Boyer–Moore**.

На моїх вимірах Boyer–Moore стабільно виграє і на “exists”, і на “fake”.
